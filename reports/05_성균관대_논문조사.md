### 4.1 핵심 관찰
1. **추천시스템(DIAL Lab, 이종욱)이 양과 질 모두 압도적**. SIGIR에 정기적으로 학생 1저자 논문 게재 → 졸업 직후 박사 임용 사례까지.
2. **NLP/LLM은 4개 연구실에 분산** (NLP Lab 고영중, HLI Lab 박진영, LearnData Lab 박호건, 정보및지능시스템 이지형). LLM 안전성/편향/디톡싱 주제 비중이 큼.
3. **Deepfake/Privacy/Unlearning은 DASH Lab(우사이먼)이 단일 연구실로 가장 많은 학생 논문 산출**. 2025년 하반기 단일 분기에 NeurIPS/CIKM/ICCV/ACM MM 8편 이상.
4. **컴퓨터 비전은 Visual Computing Lab(허재필)·Applied AI(김광수)·AIM Lab(이준호) 3축**.
5. **AI4Science / Medical AI / 강화학습 일반론**은 학생 1저자 논문 비중이 상대적으로 낮음 (강화학습은 CSI Lab이 거의 단독). 사용자가 후자 중 하나를 택할 경우 SKKU의 직접 비교 사례는 적음.
6. **학생 학위논문 자체 PDF는 dCollection 비공개·동적 페이지 한계로 본 조사에서 미수집** — RISS 직접 검증된 학위논문은 #51, #52 단 2건.

## 6. 한계 및 향후 보완 사항

### 6.1 본 조사의 한계
1. **dCollection 자동 추출 실패** — SKKU dCollection은 SPA(Single Page Application)로 검색 결과가 동적 렌더링되어 WebFetch로 학위논문 메타데이터 추출 불가. 결과 페이지가 모두 빈 템플릿으로 반환됨.
2. **RISS 학과별 필터링 0건** — `znDept,인공지능학과` 또는 `znDept,데이터사이언스융합학과`로 검색 시 매치 0건. 학과명 표기 불일치 또는 인덱싱 지연 추정. RISS의 znCreator/znPublisher 검색 결과는 1972~2001년 데이터에 편중.
3. **학위논문 vs 학회 논문 갭** — 본 조사의 "추정 학위논문 주제"는 학회 게재 논문 기반. 한국 AI 대학원 관행상 일치도가 높지만 100% 일치는 보장 못 함. 특히 학생이 여러 편 학회 게재 시 어떤 것이 "졸업 논문 본체"인지 외부 식별 불가.
4. **2023년 데이터 부족** — News 페이지 페이지네이션상 2023년 항목은 후반부에 분포하며, AAAI 2024(2023.12 게재 결정)·SIGIR 2023·CIKM 2023·ACL 2023·CVPR 2023·ICCV 2023 정보는 일부만 캡처. 2022~2023 사이 졸업 학생의 학위논문은 본 표에서 누락 가능.
5. **인공지능융합학과(xAI), 데이터사이언스융합학과의 학생 1저자 논문 정보 부족** — 인공지능학과(ai.skku.edu) 뉴스에 비해 xai.skku.edu / ads.skku.edu의 학생별 publication 페이지가 빈약하거나 비공개. 표 3.2가 표 3.1보다 짧은 이유.
6. **PDF URL 미수집** — 학위논문 PDF 링크는 dCollection의 file/download/{hash} 형태이며 본 조사에서 직접 확인된 것은 없음. 제출된 #51 김영준 학위논문도 RISS 메타데이터만 확인. 사용자가 실제 PDF 접근 시 SKKU 도서관(lib.skku.edu) 또는 RISS 기관계정 로그인 필요.
7. **수상/우수논문상 정보 일부만 캡처** — DASH Lab의 PAKDD 2024 Best Paper Running-Up, CSI Lab의 2023 Spectrum Challenge 1위, AI Grand Challenge 등 단편 정보만 수집. 학과별 "최우수 학위논문상" 같은 공식 표창 데이터는 미확보.

### 6.2 향후 보완 권장 작업
- [ ] **dCollection 직접 방문**: 사용자가 브라우저에서 https://dcollection.skku.edu 접속 → "학과별" → "일반대학원" → "인공지능학과/데이터사이언스융합학과" 트리 탐색하여 PDF URL 획득 (자동화 불가, 수동 작업 필요).
- [ ] **RISS 기관계정**: 고려대 RISS 기관 계정으로 상세검색 → 발행처 "성균관대학교" + 학과명 자유텍스트 검색 (인덱싱 누락분 직접 확인).
- [ ] **dxLab, DASH Lab, DIAL Lab Alumni 페이지** 직접 방문하여 졸업생별 학위논문 제목·dCollection 링크 수집 (lab 사이트가 공개 시).
- [ ] **2022년 이전 졸업생 패턴**: 학과 신설 직후(2019~2021) 졸업생 학위논문은 RISS에 안정적으로 등록되어 있을 가능성 → control_no 직접 수집 가능.
- [ ] **사용자의 관심 영역 확정 후** 해당 분야 SKKU 연구실(예: LLM이면 NLP Lab, RL이면 CSI Lab)의 학생 publication을 별도 심층 조사.
