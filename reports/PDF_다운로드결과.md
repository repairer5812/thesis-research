### 3.1 성공 패턴 (학내 OAK/dCollection 외부 접근 가능)

**필수 헤더**:
```
User-Agent: Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 ...
Accept: text/html,application/xhtml+xml,...
```

**dCollection 패턴 (KU/CAU/HY)**:
1. `https://{dcollection-host}/common/orgView/{ID}` 페이지 GET (쿠키 저장)
2. HTML에서 `/public_resource/pdf/{ID}_{timestamp}.pdf` 추출
3. 같은 세션 쿠키 + Referer 헤더로 PDF GET

**S-Space 패턴 (SNU)**:
1. `https://s-space.snu.ac.kr/handle/{handle}` GET
2. HTML에서 `/bitstream/{handle}/1/{filename}.pdf` 추출
3. 직접 GET

### 3.2 차단/실패 패턴

- **Yonsei dCollection**: 외부 IP 403
- **KHU dCollection**: 외부 IP 403
- **UOS dCollection**: 405 Method Not Allowed
- **Sogang dCollection**: 검색 결과가 JavaScript로 동적 렌더링
- **SKKU dCollection**: 동일 SPA 한계 + 학과 코드 부재
- **RISS [원문보기]**: 회원 로그인 + 기관계정 인증 필요
