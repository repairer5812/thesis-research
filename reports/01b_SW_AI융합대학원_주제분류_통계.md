## 3. 학과별 특성

### 빅데이터융합학과 (~25건)
- 금융/회계/재무 도메인 응용 + LLM/NLP 강함
- 대표: 재무분석 RAG, LLM 추천, 환율/주가 예측, 신용카드 부정탐지
- 지도교수: 임희석 등

### 인공지능융합학과 (AI융합학과, ~13건)
- LLM/VLM, 멀티모달, 지식증류, 의료영상
- 대표: 병리 VLM, LLM-OpenVocab3D, K-MIR, Enhanced-FastDFKD

### 소프트웨어보안학과 (~12건)
- AI 기반 보안 (DDoS, 악성코드, 취약점, ZTNA)
- 대표: LLM+DDoS, C/C++ 취약점 탐지 LM, TARA 자동화
- 지도교수: 유헌창 등

## 4. 주요 트렌드 (2024 → 2026)

- **LLM 활용 일반화**: 단순 LLM 사용 → 프롬프트 엔지니어링 → 파인튜닝 → RAG → Agent (LangGraph) → Context Engineering
- **금융/제조 도메인 응용 多**: 한국 산업 현실 반영 (제약, 반도체, 금융, 회계)
- **보안 + LLM 결합 증가**: 취약점 탐지, CTI, 정적분석에 LM 적용
- **멀티모달 출현**: 2025년부터 VLM, 3D, 멀티모달 시계열 등
- **연합학습 꾸준**: 4건, 모바일/금융/보안에 적용
