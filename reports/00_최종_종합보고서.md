## 4. 사용자 관심분야별 진로 매칭 가이드

### 4.1 LLM 우선이라면
- **사용자에게 가장 자연스러운 선택지**: 본인이 SW·AI 융합대학원 소속이고, LLM 관련 논문 비중 30%
- **참조 학교/연구실**:
  - 동일 학교 내: SW·AI 빅데이터융합학과 (RAG, Agent), 인공지능융합학과 (Multimodal LLM)
  - 외부 벤치마크: 서울대 GSDS, 서강대 AI·SW, 연세대 NLP Lab(이경호/여진영)
- **추천 졸업논문 방향 예시**: 한국어 도메인 RAG, LLM Agent의 Tool-Use 신뢰성, Domain-Specific LLM 평가 벤치마크, RLHF/Preference Optimization 응용

### 4.2 AI4Science 우선이라면
- **참조 학교/연구실**:
  - 서울대 IPAI (Weekast 기상, ECG 진단)
  - 고려대 일반대 인공지능학과 (Materials, Combinatorial Optimization)
  - 시립대 (공공 환경 데이터)
- **추천 방향 예시**: Materials-aware LM, 의료영상 + LLM Reasoning, 기상/환경 시계열 + Physics-Informed AI
- **사용자 강점**: SW·AI 융합대학원 빅데이터융합학과 출신자가 도메인 데이터를 들고 들어가서 모델링하는 흐름 자연스러움

### 4.3 RL 우선이라면
- **현실적 어려움**: SW·AI 융합대학원에는 RL 멘토가 거의 없음 (91건 중 1~2건뿐)
- **선택지**:
  - 일반대학원 인공지능학과로 일부 협업 추진 가능 (RL+Robot, RL+AI4Sci 사례 있음)
  - 협력 진학 또는 외부 인턴십
- **추천 방향**: Offline RL, RLHF/Preference, RL × Multimodal Agent

### 4.4 멀티모달 우선이라면
- **자연스러운 선택**: SW·AI 융합대학원 인공지능융합학과 (K-MIR, LLM-OpenVocab3D, VLM 시각 추론 사례 있음)
- **외부 벤치마크**: 서울대 GSDS/협동AI, 시립대(VLM)
- **추천 방향**: 한국어 멀티모달 데이터셋, VLM 경량화, RL × VLM

## 6. 사용자에 대한 종합 권장 사항

### 6.1 분야 미확정 → 분야 결정에 도움이 되는 관찰
- **현 SW·AI 융합대학원의 강점은 LLM 응용·도메인(금융/제조)·보안**
- **본교 일반대학원 인공지능학과는 학계 지향 (Speech/CV/Medical/Robot)** — 사용자가 박사 진학을 고려한다면 본교 일반대학원과 협력 또는 외부 PhD 진학 vs 산업 취업이 핵심 분기점
- **본 보고서 데이터로 본 결론**:
  - 산업 취업 + 응용 → LLM (RAG/Agent) 또는 멀티모달
  - 학계 지향 + 박사 진학 → AI4Science 또는 RL (서울대/고려대 일반대 협업 필요)

### 6.2 졸업논문 방향 추천 (Top 5)
1. **LLM × 도메인 데이터 RAG/Agent** — 본 학과 트렌드와 일치, 멘토링 강함, 산업 가치 높음
2. **Multimodal (VLM/한국어) 데이터셋 + 경량화** — K-MIR류 후속, 차별화 + 학회 발표 가능
3. **AI4Science (Materials, 의료, 환경)** — 도메인 데이터 가지고 LLM/Diffusion 응용, 기여도 명확
4. **RL × Vision (VLA)** — 멘토는 적지만 핫 트렌드, 본교 일반대학원 인공지능학과 협력 검토
5. **AI Security × LLM** — 본 학과의 강점과 LLM 결합, 박세익(2024 박세익 사례)·문광석(CTI) 논문 후속 가능

2. **우수학위논문상 정보 비공개** — 모든 학교에서 상수상 메타데이터를 RISS/dCollection에 기록하지 않음. 학과 홈페이지 게시판 별도 확인 필요.

4. **카테고리 분류는 휴리스틱** — 제목/요약 키워드 기반 자동 분류이므로 일부 오분류 가능.

6. **본 데이터 ≠ 전수** — 일부 학과(인공지능솔루션 ERICA 등 신설 학과)는 졸업생 부재로 미포함.
