## 1. 조사 범위

- **일반대학원 (Graduate School)**

## 6. 한계 및 주의사항

### 6.1 데이터 수집 한계
1. **dCollection 직접 접근 실패**: `khu.dcollection.net`의 학과별 디렉토리·검색 URL이 외부 WebFetch에서 일관되게 오류 페이지(404)를 반환. 본 조사는 RISS 통합 검색 키워드 결과에 의존하므로 dCollection 단독 등록(RISS 미연동) 논문이 누락 가능.
2. **발행기관 facet 미확보**: RISS facet (학과별 건수 분포) 정보를 외부 페이지 렌더링에서 일관 추출 불가. 학과 분포는 본문 표시 학과명에 의존.
3. **인공지능학과 분리 미확인**: 경희대 학부·대학원 구조상 "인공지능학과"는 컴퓨터공학부 산하 학부 트랙으로 존재. 일반대학원 학위논문 발행기관으로 별도 분리되어 RISS에 인덱싱된 사례 없음 (본 조사 시점 기준). 컴퓨터공학과·소프트웨어융합학과 학위논문이 사실상 인공지능학과 학생 논문을 포함.
4. **중복 제거 불완전**: 동일 논문이 키워드별로 중복 검색됨. 제목·저자 기반으로 best-effort 중복 제거했으나 완전하지 않을 수 있음.
5. **지도교수·초록·키워드 미수집**: 본 단계는 제목·저자·학과·연도까지만. 추천 Top 10에 대한 PDF·초록 확보는 후속 단계 필요.

### 6.2 누락 가능성이 큰 영역
- 경희대 국제캠퍼스 소프트웨어융합대학 학부생이 진학한 "데이터사이언스학과" 등 신설학과 (RISS 인덱싱 지연)
- 비공개 학위논문 (저자 동의 미확인 시 RISS에서 제목만 노출되거나 누락)
- 2026년 2월 졸업분 중 4월 시점 RISS 미동기화 논문

### 6.3 후속 작업 권장
1. **dCollection 직접 방문 확인**: `https://khu.dcollection.net`에서 학과 브라우즈 (대학원 → 컴퓨터공학과 / 소프트웨어융합학과)로 정확한 publisher 단위 카운트 확보.
2. **Top 10 PDF 다운로드**: RISS의 각 논문 상세 페이지에서 원문 PDF 링크 확보 → `PDF/` 폴더 저장.
3. **김태성·태인규·오지헌 논문 초록 확보**: 사용자 관심분야 결정에 직접 활용.
4. **지도교수 매핑**: 컴퓨터공학과 연구실(`https://ce.khu.ac.kr/ce/user/contents/view.do?menuNo=1600077`)과 매칭하여 진학 가능 연구실 식별.
